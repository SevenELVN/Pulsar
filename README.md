# Pulsar
Pulsar позволяет обмениваться текстовыми сообщениями с полным сквозным шифрованием (E2E), используя любой сервер Icecast/Shoutcast в качестве слепого ретранслятора. Ввод текста возможен как с физических кнопок азбукой Морзе на ESP32, так и с экранной клавиатуры Android-клиента. Связь анонимна и не может быть прочитана даже администратором сервера.
